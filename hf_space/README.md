# 🛸 ConflictEnv: Strategic AI Agent

An Elite Executive Assistant trained with **GRPO (Reinforcement Learning)** to resolve high-stakes scheduling conflicts.

## What it does
- Analyzes complex calendar conflicts (Work vs. Life vs. Social)
- Uses deep `<thought>` reasoning before acting
- Outputs structured JSON commands (delegate, reschedule, cancel)

## Model
- **Base**: Qwen 2.5 1.5B Instruct
- **Training**: GRPO on 5,000+ synthetic conflict scenarios
- **Hosted on**: [purvansh01/conflict-env-final](https://huggingface.co/purvansh01/conflict-env-final)
